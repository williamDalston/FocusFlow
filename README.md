# 7-Minute Workout iOS App

A beautiful, native iOS workout app featuring the scientifically-backed 7-minute workout routine. Built with SwiftUI for iPhone and Apple Watch.

## Features

- **12 High-Intensity Exercises**: Complete the full 7-minute workout routine
- **Beautiful UI**: Modern SwiftUI interface with circular progress indicators
- **Sound & Haptics**: Audio feedback and vibration for workout transitions
- **Progress Tracking**: Track your workout history, streaks, and statistics
- **Apple Watch Support**: Workout companion app for Apple Watch
- **Settings**: Customizable sound, vibration, and appearance settings

## Project Structure

```
workout-app/
├── OneTapGratitude.xcodeproj/    # Xcode project file
├── OneTapGratitude/              # Main iOS app code
│   ├── Workout/                  # Workout engine and views
│   ├── Models/                   # Data models
│   ├── UI/                       # UI components
│   └── System/                   # System utilities
└── OneTapGratitudeWatch/         # Apple Watch app
```

## Opening in Xcode

1. Open `workout-app/OneTapGratitude.xcodeproj` in Xcode
2. Select your development team in Signing & Capabilities
3. Build and run on your device or simulator
